# AI Context File

This repository contains a multi-service fintech platform called Continuum.

The purpose of the platform is financial information continuity.

The system stores structured financial information so nominees can reconstruct a user's financial life after death.

The platform does not transfer funds or store credentials.

Important design principles:

1. Security first
2. No plaintext sensitive data
3. No password storage
4. No financial transaction execution
5. Clear audit logs
6. Strong context isolation
