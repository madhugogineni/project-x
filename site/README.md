# Project X Site

Public marketing website for Project X, the financial continuity platform.

## Stack

- Next.js App Router
- React
- TypeScript

## Getting Started

```bash
npm install
npm run dev
```

The site runs on `http://localhost:3020` in the local Docker workflow.

## Environment

Copy `.env.example` to `.env` and adjust values as needed.

## Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
```
