# Continuum Site

Public marketing website for Continuum, the financial continuity platform.

## Stack

- Next.js App Router
- React
- TypeScript

## Getting Started

```bash
npm install
npm run dev
```

The app runs on `http://localhost:3000`.

## Environment

Copy `.env.example` to `.env.local` and adjust values as needed.

## Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
```
