# Project X App

Primary application for onboarding users, managing assets, and coordinating nominee access.

## Stack

- Next.js App Router
- React
- TypeScript

## Getting Started

```bash
npm install
npm run dev
```

The app runs on `http://localhost:3021` in the local Docker workflow.

## Environment

Copy `.env.example` to `.env` and point `NEXT_PUBLIC_API_BASE_URL` to the backend.

## Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
npm run test:ui
```
